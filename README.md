# Belajar Golang Dasar bersama Programmer Zaman Now

## How to run?

```
$ go run ./hello-world.go
```

## How to compile?

```
$ go build ./hello-world.go
$ ./hello-world.go
```

## Init Go Mod

```
$ go mod init project-name
```
